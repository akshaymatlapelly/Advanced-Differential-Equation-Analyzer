# 🚀 Advanced Differential Equation Analyzer

## 📖 Overview

Advanced Differential Equation Analyzer is a Python-based mathematical tool designed to analyze and classify differential equations. The application provides symbolic representations of Jacobian and Hessian matrices, computes integrating factors, detects equation order and degree, and maintains an analysis history log.

This project was developed using pure Python without external libraries, making it lightweight, educational, and easy to understand for students learning differential equations and mathematical computing.

---

## ✨ Features

### 🔍 Differential Equation Analysis

* Classifies differential equations
* Identifies Linear Differential Equations
* Identifies Bernoulli Differential Equations
* Identifies Exact Differential Equations
* Identifies General Differential Equations

### 📊 Equation Properties

* Detects Order of Differential Equations
* Detects Degree of Differential Equations

### 🧮 Mathematical Tools

* Integrating Factor Generator
* Jacobian Matrix Generator (Symbolic Representation)
* Hessian Matrix Generator (Symbolic Representation)

### 📝 History Tracking

* Automatically stores analysis results
* Maintains a history log in `history.txt`


Navigate to the project folder:

